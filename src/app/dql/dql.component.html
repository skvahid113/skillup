The SELECT statement in SQL is used to retrieve data from one or more tables.

SELECT

  • column1, column2, ... -- List of columns to select
  • FROM table_name -- Table to retrieve data from

Examples:

SELECT first_name, last_name FROM employees;

SELECT product_name, price FROM products WHERE category_id = 1;

SELECT order_id, order_date, total_amount FROM orders;

SELECT product_name, price FROM products ORDER BY price DESC;

SELECT employee_name, hire_date FROM employees ORDER BY hire_date ASC;

The FROM statement specifies the table or tables from which data is retrieved.

FROM

  • table_name -- Name of the table

Examples:

FROM customers;

FROM employees e JOIN departments d ON e.department_id = d.department_id;

The GROUP BY statement is used to group rows that have the same values into summary rows.

GROUP BY

  • column1, column2, ... -- Columns to group by

Examples:

SELECT department, AVG(salary) FROM employees GROUP BY department;

SELECT category, COUNT(*) FROM products GROUP BY category;

The HAVING statement is used to filter the results of a GROUP BY clause.

HAVING

  • aggregate_function(column) condition; -- Condition to filter grouped data

Examples:

SELECT department, AVG(salary) FROM employees GROUP BY department HAVING AVG(salary) > 50000;

SELECT category, COUNT(*) FROM products GROUP BY category HAVING COUNT(*) > 5;

The ORDER BY statement is used to sort the result set in ascending or descending order.

ORDER BY

  • column1 ASC/DESC, column2 ASC/DESC, ... -- Columns to sort by

Examples:

SELECT product_name, price FROM products ORDER BY price DESC;

SELECT employee_name, hire_date FROM employees ORDER BY hire_date ASC;
