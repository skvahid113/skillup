ISNULL

  • ISNULL (Expression, Value): Syntax: ISNULL(expression, value)

    Description: The ISNULL function returns the specified value if the expression is NULL. It is often used to handle NULL values gracefully by providing a default value.

    SQL Example: 
    
SELECT ProductName, ISNULL(DiscountedPrice, RegularPrice) AS Price
FROM Products;
    Output: 
    
| ProductName    | Price     |
| --------------- | --------- |
| Product 1       | 10.00     |
| Product 2       | 15.00     |
| Product 3       | 8.00      |
| Product 4       | 20.00     |
  • ISNULL (Expression, 'N/A'): Syntax: ISNULL(expression, 'N/A')

    Description: In this example, we use 'N/A' as the default value to replace NULL values.

    SQL Example: 
    
SELECT CustomerName, ISNULL(PhoneNumber, 'N/A') AS Contact
FROM Customers;
    Output: 
    
| CustomerName   | Contact   |
| --------------- | --------- |
| Customer 1      | 123-4567  |
| Customer 2      | N/A       |
| Customer 3      | 987-6543  |

IS Null

  • IS Null (Column): Syntax: Column IS NULL

    Description: The IS NULL condition checks if a column's value is NULL. It is commonly used in WHERE clauses to filter rows with NULL values.

    SQL Example: 
    
SELECT EmployeeName
FROM Employees
WHERE Supervisor IS NULL;
    Output: 
    
| EmployeeName    |
| --------------- |
| John            |
| Alice           |
| Bob             |
  • IS Null (Status): Syntax: Status IS NULL

    Description: In this example, we filter records where the 'Status' column is NULL, indicating unprocessed data.

    SQL Example: 
    
SELECT OrderID, OrderDate
FROM Orders
WHERE Status IS NULL;
    Output: 
    
| OrderID    | OrderDate |
| ---------- | ---------- |
| 1001       | 2023-01-15 |
| 1004       | 2023-02-10 |
| 1007       | 2023-03-05 |

COALESCE

  • COALESCE (Column1, Column2, ...): Syntax: COALESCE(column1, column2, ...)

    Description: The COALESCE function returns the first non-NULL value from a list of columns. It is useful for selecting the first available value from multiple options.

    SQL Example: 
    
SELECT ProductName, COALESCE(PromotionalPrice, DiscountedPrice, RegularPrice) AS Price
FROM Products;
    Output: 
    
| ProductName    | Price     |
| --------------- | --------- |
| Product 1       | 8.00      |
| Product 2       | 15.00     |
| Product 3       | 12.00     |
| Product 4       | 20.00     |
  • COALESCE (FirstName, LastName, 'N/A'): Syntax: COALESCE(FirstName, LastName, 'N/A')

    Description: In this example, we use 'N/A' as the default value if both 'FirstName' and 'LastName' are NULL.

    SQL Example: 
    
SELECT EmployeeID, COALESCE(FirstName, LastName, 'N/A') AS FullName
FROM Employees;
    Output: 
    
| EmployeeID | FullName |
| ---------- | -------- |
| 101        | John     |
| 102        | Alice    |
| 103        | Bob      |
| 104        | N/A      |

NULLIF

  • NULLIF (Value1, Value2): Syntax: NULLIF(value1, value2)

    Description: The NULLIF function compares two values and returns NULL if they are equal; otherwise, it returns the first value. It is often used to handle cases where you want to treat specific values as NULL.

    SQL Example: 
    
SELECT EmployeeName, NULLIF(Supervisor, 'None') AS RealSupervisor
FROM Employees;
    Output: 
    
| EmployeeName    | RealSupervisor |
| --------------- | -------------- |
| John            | John           |
| Alice           | Bob            |
| Bob             | Bob            |
  • NULLIF (Score1, Score2): Syntax: NULLIF(Score1, Score2)

    Description: In this example, NULLIF is used to compare two test scores and return NULL if they are the same.

    SQL Example: 
    
SELECT StudentName, NULLIF(TestScore1, TestScore2) AS BestScore
FROM Students;
    Output: 
    
| StudentName    | BestScore |
| --------------- | --------- |
| Alice           | 98        |
| Bob             | NULL      |
| Carol           | 85        |
